# HighView Student Engagement Platform

A comprehensive student analytics and engagement platform built with React, TypeScript, and Vite. Winner of AWS Hackathon.

## 🎯 Core Features

### ✅ Implemented Features
- **📊 Attendance Tracking** - Real-time check-in/check-out with status monitoring
- **🏆 Student Leaderboard** - Rankings with multi-metric scoring system
- **📈 Analytics & Reporting** - Comprehensive data visualization and insights
- **📱 Live Dashboard** - Real-time session monitoring and engagement metrics
- **👥 Student Management** - Complete student profile and performance tracking
- **📚 Course Management** - Course performance and enrollment analytics
- **🎓 Session Management** - Live session tracking and activity monitoring

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Shadcn/ui** - UI components
- **React Router DOM** - Routing
- **Recharts** - Data visualization
- **Framer Motion** - Animations
- **Lucide React** - Icons

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
